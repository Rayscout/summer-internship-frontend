The Introduction about HTML & CSS & JavaScript

HTML & CSS & JavaScript 简介

Author: Ray Scout | Date: August 7, 2025

1. HTML (HyperText Markup Language)

EN: HTML is the standard markup language for documents designed to be displayed in a web browser. It forms the structure of a web page, like the skeleton of a human body.

中： HTML（超文本标记语言）是用于创建网页的标准标记语言。它构成了网页的结构，就像人体的骨架。 

<!-- Example of a simple HTML structure -->
<!DOCTYPE html>
<html>
<head>
  <title>My Page</title>
</head>
<body>
  <h1>Hello, World!</h1>
  <p>This is a paragraph.</p>
</body>
</html>

2. CSS (Cascading Style Sheets)

EN: CSS is the language we use to style an HTML document. It describes how HTML elements should be displayed, handling the layout, colors, and fonts. It's like the clothing and appearance of the body.

中： CSS（层叠样式表）是我们用来设计HTML文档样式的语言。它描述了HTML元素应如何显示，负责处理布局、颜色和字体。它就像人体的衣服和外貌。 

/* Example of CSS styling a paragraph */
p {
  color: #007aff; /* Sets the text color to Apple's blue */
  font-size: 17px;
  line-height: 1.65;
}

3. JavaScript (JS)

EN: JavaScript is a programming language that enables you to create dynamically updating content, control multimedia, animate images, and much more. It makes web pages interactive, like making a button navigate to another site.

中： JavaScript 是一门编程语言，能让你创建动态更新的内容、控制多媒体、制作图像动画等。它让网页变得可交互，比如让一个按钮跳转到另一个网站。
